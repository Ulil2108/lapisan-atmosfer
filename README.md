# 🌍 Media Pembelajaran Lapisan Atmosfer

Media pembelajaran interaktif untuk siswa SMA tentang lapisan-lapisan atmosfer Bumi menggunakan teknologi web modern (HTML, CSS, JavaScript).

## ✨ Fitur Utama

### 🎯 Visualisasi Interaktif
- **Bumi Dinamis** - Representasi visual 3D dengan gradien warna realistis
- **Astronot Animasi** - Karakter yang bergerak naik-turun mengikuti lapisan atmosfer
- **5 Lapisan Atmosfer** - Visualisasi berlapis dengan warna gradient yang berbeda:
  - 🔵 **Troposfer** - Lapisan cuaca (0-17 km)
  - 🟦 **Stratosfer** - Lapisan ozon (17-50 km)  
  - 🟪 **Mesosfer** - Lapisan terdingin (50-85 km)
  - 🔴 **Termosfer** - Lapisan panas (85-500 km)
  - 🟣 **Eksosfer** - Lapisan transisi (>500 km)

### 🎮 Kontrol Pengguna
- **Tombol Naik (⬆️)** - Naik ke lapisan lebih tinggi
- **Tombol Turun (⬇️)** - Turun ke lapisan lebih rendah
- **Keyboard Support** - Gunakan Arrow Up/Down untuk navigasi
- **Smart Button Logic** - Tombol otomatis disable di batas (permukaan bumi & eksosfer)

### 📚 Informasi Lengkap
Setiap lapisan memiliki:
- Nama dan ketinggian
- Karakteristik suhu
- Deskripsi singkat
- Informasi detail yang relevan

### 🎨 Desain Responsif
- Tampilan sempurna di desktop, tablet, dan mobile
- Gradient background yang menarik
- Animasi smooth dan fluid
- Interface yang user-friendly

## 🚀 Cara Menjalankan

### Opsi 1: Langsung di Browser
1. Download atau clone repository ini
2. Buka file `index.html` di browser favorit Anda
3. Mulai jelajahi lapisan atmosfer!

### Opsi 2: Menggunakan Live Server (VS Code)
1. Clone repository ke folder lokal
2. Buka folder di VS Code
3. Install extension "Live Server"
4. Klik kanan di `index.html` → "Open with Live Server"
5. Browser akan membuka otomatis

## 📖 Penjelasan Lapisan Atmosfer

### 1️⃣ Troposfer (0 - 17 km)
- **Karakteristik**: Lapisan tempat terjadi cuaca dan aktivitas hidup
- **Suhu**: Menurun seiring ketinggian
- **Konten**: 80% massa udara atmosfer, mengandung awan, hujan, dan angin

### 2️⃣ Stratosfer (17 - 50 km)
- **Karakteristik**: Melindungi Bumi dari radiasi UV
- **Suhu**: Meningkat seiring ketinggian
- **Keunikan**: Mengandung lapisan ozon (O₃) yang vital

### 3️⃣ Mesosfer (50 - 85 km)
- **Karakteristik**: Lapisan terdingin di atmosfer
- **Suhu**: Bisa mencapai -90°C
- **Fenomena**: Tempat meteor terbakar menciptakan bintang jatuh

### 4️⃣ Termosfer (85 - 500 km)
- **Karakteristik**: Lapisan sangat panas
- **Suhu**: Ribuan derajat Celsius
- **Fenomena**: Aurora borealis terjadi, satelit mengorbit

### 5️⃣ Eksosfer (>500 km)
- **Karakteristik**: Lapisan paling luar, transisi ke ruang angkasa
- **Konten**: Udara sangat jarang dan jarang ditemukan molekul
- **Fungsi**: Tempat satelit buatan dan stasiun luar angkasa
